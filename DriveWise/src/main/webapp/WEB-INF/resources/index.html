혼잡지표 실시간 데이터를 수집하여 도로별 혼잡 정보를 제공합니다.
안전지표 도로의 상태 및 사고 발생 지표를 반영하여 도로별 안전지수를 시각화된 자료로 제공합니다.
최적 경로 안내 도로의 혼잡·안전수준과 소요시간을 고려해 최적의 경로를 안내합니다.
Solution

스마트 교통 솔루션은 실시간 데이터와 머신러닝 기술을 활용하여, 교통 문제를 해결하고 쾌적한 이동 환경을 제공해 스마트시티 구현에 기여합니다.

01

교통혼잡지수 분석

교통·기상·도로 정보가 혼잡지수에 끼치는 영향을 분석하여 실시간 도로별 혼잡지수를 산출합니다.

02

교통안전지수 분석

교통·도로·시간 정보가 교통안전지수에 끼치는 영향을 분석하고 교통안전지수를 산출합니다.

03

예상 통행시간 분석

시간·교통·기상 데이터가 도로 통과시간에 끼치는 영향을 분석해 도로 통행시간 예측 모델을 도출합니다.

04

최적 경로 안내

혼잡·안전지수와 예상 통행시간을 바탕으로 실시간 경로평가지수를 생성하여 최적의 경로를 안내합니다.
